# Whos-not-your-follower

# Requirements / Dependencies
* python version above ```3.6```
* selenium
  ```
  > pip install selenium
  ```
* Chrome webdriver [ [download-link](https://chromedriver.chromium.org/) ]
* beautifulsoup4
  ```
  > pip install beautifulsoup4
  ```
* requests
  ```
  > pip install requests
  ```
